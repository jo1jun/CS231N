아래 계산 그래프들은 node들을 합쳐 더 간단하게 표현하고 더 효율적으로 계산할 수 있다. 
계산그래프 전공은 아니기에 더 단순화 시키진 않았다. 추후에 필요하다면 수정할 것.
아래 계산그래프를 기반으로 위의 code들을 구현하였다.
처음 학습 과정에서 명확한 이해를 위해 그려볼 순 있지만 이미 잘 구현된 framwork 들을 활용하면 된다.
